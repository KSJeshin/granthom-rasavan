## Toolchain Options

### Option 1 — Microsoft Keyboard Layout Creator (MSKLC)
- Free tool from Microsoft
- Download: https://www.microsoft.com/en-us/download/details.aspx?id=102134
- Produces a standard Windows `.klc` source file and a compiled installer
- Installs as a native Windows keyboard layout — appears in Language Settings
- Limitation: supports up to 4 layers (Normal, Shift, AltGr, AltGr+Shift) — matches Granthom Rasavan exactly
- Caps Lock reassignment requires a registry edit or AutoHotKey script alongside

### Option 2 — Keyman
- Free and open source
- Download: https://keyman.com
- More flexible than MSKLC — supports complex input logic
- Better suited if the layout grows to include IME-style features
- Produces a `.keyman-package` file distributable independently

**Recommendation for initial release:** MSKLC — simpler, produces a native Windows installer, no dependency on third-party software for end users.

## How to Install (once file is available)

**MSKLC installer:**
1. Run `granthom_rasavan_setup.exe`
2. Go to Settings → Time & Language → Language → Keyboard
3. Add **Malayalam — Granthom Rasavan**
4. Switch using Win + Space or the language bar

**Keyman package:**
1. Install Keyman from https://keyman.com
2. Open the `.keyman-package` file — Keyman installs it automatically
3. Select the layout from the Keyman toolbar

## Caps Lock Behaviour on Windows

MSKLC does not natively support reassigning Caps Lock to a language toggle.
Two options for the Windows implementation:

- **AutoHotKey script** — maps Caps Lock to switch between Malayalam and English layouts at the OS level
- **Keyman** — supports Caps Lock remapping natively within the package

This will be decided during implementation. See
**Issue 05** in [issues/open_issues.md](../../../issues/open_issues.md).

## Malayalam Unicode Rendering

Windows 8.1 and later include Uniscribe and DirectWrite support for Malayalam Unicode.
The layout outputs standard Unicode sequences — no custom font or renderer required.
Recommended fonts: Noto Sans Malayalam, Rachana, Meera.
